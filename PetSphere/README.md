## Getting Started

To get started with the PetSphere app, follow these steps:

1. **Clone the repository**:
   ```
   git clone https://github.com/habib-153/Pet_Cohort.git
   cd PetSphere
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the app**:
   ```
   expo start
   ```
